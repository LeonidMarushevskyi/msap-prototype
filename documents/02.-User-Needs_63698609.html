Mississippi Agile Pool : 02. User Needs

Created by Leonid Marushevskiy, last modified by Pavel Khozhainov on Oct 10, 2016

Step I (raw search)

Parent - find the Child Care facility for own kid(s)

Criteria:

Openings exists - basic requirement

Location and Hours - minimize commute

Special Needs if any - services must be provided

Quality (Stars, Languages, User reviews) - nice to have

Price - stay in some reasonable budget

Foster Parent - work with the Caseworker to find child care for foster kid(s) (and own)

Criteria:

Openings exists - basic requirement

Location and Hours - minimize commute

Special Needs if any - services must be provided

Price - stay in some reasonable budget

Caseworker - find child care for children

Criteria:

Openings exists - basic requirement

Safety for the child - clear history of the facility

Special Needs if any - services must be provided

Location and Hours - minimize commute for parent

Price - stay in some reasonable budget for parent

Extended case - need child care for more than one child

Step II (search result usage)

Caseworker  - easy communication between parent (foster or bio) and child care provider

Foster Parent - easy communication with Caseworker around possible child care providers

Caseworker/Parent - export information about all Facilities found in PDF form
