  • Leonid Marushevskiy - Functional features which need to be changed:
    1. We have three roles: Parent, Foster Paren, Caseworker. For Foster Parent and Caseworker we shall have one more menu item "Inbox" with messaging functionality. 
      For foster parentrole we shall show a button "Ask Caseworker" and switch to Inbox on click
    2. Price (per week): please think how we shall implement this control. We have price range min-max which is tied to an age group. 
    3. Main provider information: add ability to see the number of reviews and ability to see reviews
    4. Provider Details: please change this area to include following information: 
      1. Open slots for each age group
      2. Provider schedule in form day of the week - open hour - close hour
      3. Special needs which this provider accepts
      4. List of substantiated Allegations in a form: Description, date, Severity (A, B)
      5. License type
      6. Provider Capacity
      7. Provider Description (free form text)
      8. Number of Complains
      9. Number of visits
      10. Date of last Visits
    5. Show all filters: 
      1. Working hours: replace current control with 4 checkboxes: Open Overnight, Full Day, Before School, After School
      2. Available spots: please think about this control. Available spots are related to age groups. If we already defined age groups we shall be able to specify the number of available spots for this particular age group. 
      3. Please add additional filter: License type
      4. Please think about additional filter which will let user select days of the week when provider will provide their services. User will be able to check the days which he is looking for
      5. Please add additional filters: checkbox "Nocomplains", "No Allegations"
