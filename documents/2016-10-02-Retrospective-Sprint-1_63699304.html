Retrospective

What did we do well?

    • Design stage was done quickly and we were able start application development early.
    • We were able to implement a functional application prototype during the first Sprint.
    • We were able to set up continuous integration for the application,  which helps a lot in development.

What should we have done better?

    • We shall not expand the scope during the Sprint.
    • One-week Sprint was not enough to finish the development of most of the user stories. We moved them to the next Sprint. We shall develop user stories more granularly, such that we will able to close them during one Sprint.

Actions

  • Leonid Marushevskiy - Do the Sprint planning and task decomposition more carefully in order to eliminate the situation when user story is not finished in one Sprint.
